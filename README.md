# Physics Informed 2D N-Body Simulation and Analysis

Interactive N-body gravitational simulator with ML-based stability prediction.

## Features
- Newtonian N-body simulation
- FastAPI backend
- Real-time interface
- Machine learning stability classifier

## 1. Install Dependencies and Setup
```bash
pip install -r requirements.txt

python -m venv myenv
myenv\Scripts\activate
```

---

## 2. Run Setup
```bash
uvicorn backend.main:app --reload
```
